# jaa.ee Homepage

This is the homepage for NPO Improgrupp Jaa! ([http://jaa.ee](http://jaa.ee)).

## Requirements

* Docker

## Installation

Run the image:

```
docker run -p 80:80 improvjaa/jaa-ee
```

This will start a Nginx webserver image listening on port 80.
